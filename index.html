Aprendiendo Git

Aprendiendo a usar la tecnologia Git

Acciones basicas

Inicializar repositorio

Para poder usar todas las funcionalidades de git, primero debemos inicializarlo, para eso tenemos que pararnos en el directorio que vamos a querer controlar sus versiones. Abrir una termina git, y escribir la siguiente linea. Para poder ver la carpeta creada, habilitar los archivos ocultos mediante el explorador de archivos.

-git init

Guardar version de forma local

Subir repositorio a github

Una vez tengamos hecho el commit para guardar el directorio de forma local, el siguiente paso es subir el proyecto a la nube, en este caso github. Para esto tenemos que establecer la ruta con el comando "remote", y con el comando "push" podemos realizar la subida de archivos al repositorio.

-git remote add origin

-git push origin master #ulr-repositorio

Clonar repositorio
